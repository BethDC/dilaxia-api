### Setup su Eclipse
Installare Eclipse Enterprise Edition,
aggiungere JBoss Tools dall'Eclipse Marketplace (help -> marketplace),
importare il progetto su Eclipse,
premere il testo run e fare il setup con Wildfly ultima versione
